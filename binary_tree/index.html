Concept
Binary Search
Binary search is an algorithm for finding an element in a binary search tree.
Binary Tree
A data structure
In this structure, each node has up to 2 children
Binary Search Tree
A binary search tree is a binary tree where the left and right subtrees are also binary search trees.

All the nodes in the left subtree are less than the root's element, and all the elements of all the nodes in the right subtree are greater than the root's element.
Importance
With thousands of data points, sorting and finding the right data can be time consuming if not done properly. One of the most important and simplest data structures that help with this is the Binary Search Tree (BST).

If the patterns you write in your code are not efficient, you are stealing time from your users and in essence, draining them of their precious life force.
